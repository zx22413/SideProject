# 田中太郎 — 人生紀實整合建議

> **依據**：[[田中太郎_人生紀實]]、[[../00-核心企劃/當前專案_靈魂食堂/田中太郎_橫向與五味結構]]  
> **目的**：將人生紀實中的具體片段整合進遊戲劇本，依橫向軸、話題、料理、刺激類型對應。

---

## 一、整合策略

### 1.1 三種整合方式

| 方式 | 說明 | 範例 |
|------|------|------|
| **記憶劇場** | 直接引用紀實片段，以 Flex Message 卡片呈現 | Day 1 熱茶 → 【深夜閣樓】小小的手捧茶 |
| **對話台詞** | 將紀實中的對話或內心獨白融入角色台詞 | Day 2「那個夢」→ 「她笑起來...像蜜糖...」 |
| **敘事插敘** | 在記憶劇場前後加入紀實中的場景描述 | Day 3 告別 → 「他想起女兒小時候的樣子。那個舉著畫跑進工房的小女孩。」 |

### 1.2 對應原則

- **橫向軸**：每個紀實片段對應 1-2 條橫向軸（A-E）
- **刺激類型**：依料理的刺激類型（撫慰/宣洩/清醒/收斂/根源）選擇對應的紀實片段
- **五味傾向**：紀實片段的「情感調性」需與該話題/料理的五味傾向一致

#### 片段 1：便當太甜（第六章）
```
【記憶劇場 - 蜜汁燉菜版 - 擴充】

【場景：工房，傍晚】

美雪（七歲，舉著便當盒）：
「爸爸！我做了便當給你！」

他（正在量客人尺寸，頭也不抬）：
「放那邊吧。」

美雪（站在旁邊等）：
「⋯⋯」

【時間流逝】

客人走了之後，他拿起便當，打開，吃了一口飯糰。

美雪（緊張）：
「怎麼樣？」

他（咀嚼，皺眉）：
「有點太甜了。你放太多糖了。」

美雪（低下頭）：
「⋯⋯喔。」

他（放下便當）：
「下次少放一點。爸爸要繼續工作了。」

【旁白】
美雪想說：「老師說要做給最重要的人，所以我做給你。」
美雪想說：「我放糖是因為聽說甜的東西會讓人開心。」
美雪想說：「你可不可以⋯⋯多吃一點？」

但她什麼都沒說。

她只是轉身，走出工房。

那天晚上，他加班到很晚。他想起那個便當，走到桌邊——

便當盒裡，還剩下大半。

他拿起筷子，把剩下的全部吃完。

其實很好吃。甜甜的，像美雪的笑容。

但他沒有告訴她。
```

**整合位置**：Day 2 做蜜汁燉菜 → 在「又甜又鹹」對話後，追加此記憶劇場，強化 C 美雪軸。

#### 片段 2：美雪畫「爸爸的店」（第六章）
```
【記憶劇場 - 蜜汁燉菜版 - 另一段】

【場景：工房，美雪五歲】

美雪（舉著畫，雀躍）：
「爸爸！你看！我畫了你的店！」

畫的是「針尖工房」——歪歪扭扭的房子，門口站著一個小人，旁邊寫著「爸爸的店」。

他（正在趕急單，抬頭看了一眼）：
「嗯，畫得不錯。爸爸要工作了，你自己去玩吧。」

美雪（笑容僵在臉上）：
「⋯⋯嗯。」

她低下頭，抱著畫走了出去。

【旁白】
他不知道的是，美雪在門口站了很久，等著他叫她回來。

他沒有。

那張畫後來被美雪夾在日記本裡，再也沒有拿出來過。

**整合位置**：Day 2 做蜜汁燉菜 → 若本輪點亮 C 美雪軸（例如 Day 1 選「沉默陪伴」），可插入此片段。

#### 片段 3：入學典禮（第六章）
```
【記憶劇場 - 蜜汁燉菜版 - 關鍵片段】

【場景：高中入學典禮】

美雪（坐在位子上，轉頭看向門口）：
「⋯⋯」

她等了五分鐘。十分鐘。

還是沒有人。

班導師（輕聲）：
「美雪同學，你爸爸⋯⋯可能塞車了吧？要不要老師先幫你別？」

美雪（低下頭）：
「好。」

【場景：工房，傍晚】

美雪（推開門）：
「爸爸。」

他（抬頭）：
「回來了啊。典禮怎麼樣？」

美雪（愣住）：
「你⋯⋯忘記了？」

他（看日曆，臉色變了）：
「我⋯⋯對不起。這件西裝明天要交，我⋯⋯」

美雪（聲音發抖）：
「那件西裝很重要嗎？」

他：
「是首相要穿的⋯⋯」

美雪（眼淚掉下來）：
「比我重要嗎？」

【沉默】

美雪（轉身跑出去）：
「媽媽說對了。你只會用東西表達。但我不是衣服，爸爸。我是你的女兒。」

【旁白】
從那天起，父女之間就有了一道看不見的牆。

**整合位置**：Day 2 做蜜汁燉菜 → 若本輪收集「眼淚」標籤（「那個夢」Part 3），可插入此片段，強化 C 美雪軸的「裂痕」。
