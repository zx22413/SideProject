## 📅 每日循環結構（儀式化設計）

### 時段系統：模擬真實經營

```

→ 客人反應
→ 記憶劇場（如果料理正確）
→ 黑貓簡短評論
→ 次日循環

```

## 🧩 記憶即食材系統

### 核心概念：「你用他的記憶，做出他的料理」

#### 食材分類

**基礎食材**（一開始就有）：
- 水、米、麵、蔬菜（物理食材）

**記憶食材**（透過對話解鎖）：
- 抽象概念的具象化

#### 解鎖機制範例

```
【對話中】

田中：「好冷...我的手...動不了...」

━━━━━━━━━━━━━━━

💧 [寒冷] 
→ 廚房解鎖：溫暖屬性的食材

━━━━━━━━━━━━━━━

      一根針...銀色的針...」

━━━━━━━━━━━━━━━

[系統提示]
新的記憶碎片...

🪡 [針]
→ 廚房解鎖：針（抽象食材）
→ 用途：「縫合破碎的記憶」

━━━━━━━━━━━━━━━
```

### 最終料理（V4.9 動態化）

Day 3 的料理名稱、描述與演出依**五味結算**自動決定，與程式一致：

| 五味傾向 | 料理名稱 | 描述（程式用） |
|---------|----------|----------------|
| 🍬 甜味過重 | **糖霜幻景拼盤** | 被厚重糖粉覆蓋的雜亂拼盤。像雪一樣白，像夢一樣甜。 |
| 🦴 苦味過重 | **千針冷骨湯** | 像針一樣銳利的魚骨，泡在冰冷的黑湯裡。每一口都是痛。 |
| 🐟 平衡 | **百味蜜汁炙燒魚** | 甜鹹交織，佐以人生百味。雖然有刺，但肉質鮮美。 |

**結局判定邏輯**（`determineEnding`）：
- 甜味 > 其他四味總和 → ENDING_SWEET（糖霜幻景拼盤）
- 苦味 > (甜味 + 鹹味) → ENDING_BITTER（千針冷骨湯）
- 否則 (甜+鹹) ≥ (苦+酸) → ENDING_BALANCED（百味蜜汁炙燒魚）
- 預設：ENDING_BITTER

**遺物**（依結局類型）：
- 苦味結局：彎曲的縫紉針
- 甜味結局：泛黃的照片
- 平衡結局：銀頂針（*For my dearest Miyuki*）

**V4.6 結局變體**：結局名稱、黑貓評論、遺物描述依「變體記憶」（如缺席的典禮、失語、空蕩的店、銀座的驕傲）有 18 種組合；全收集時有專屬對話與名稱。

### 達成的設計目標

✅ **去說教化**
- 黑貓不再長篇解釋，只給提示

- 對話中的關鍵詞 → 自動解鎖食材
- 料理變成「用他的記憶做菜」

✅ **儀式化循環**
- 夜晚→白天→傍晚→深夜
- 觀察→對話→料理→揭露

✅ **延遲真相**
- Day 1: 玩家和田中都困惑
- Day 2: 暗示開始浮現
- Day 3: 完全揭露「原來已死」

✅ **神秘感保留**
- 開場極簡，不破壞懸疑
- 黑貓神秘而非全知
- 窗外的虛空始終是謎

## 🎯 V4.4 更新說明：五味系統與縱向延伸話題

### 五味系統（Flavor System）

**核心機制**：
- 每個記憶標籤對應五味數值（甜/酸/苦/辛/鹹）
- 玩家收集的記憶會累加五味數值
- 結局判定依據五味比例自動計算

**五味映射**：
- 🍯 **甜**（土）：撫慰與接地 → 適用驚魂未定的靈魂
- 🍋 **酸**（木）：聚焦與收斂 → 適用衝動決策導致後悔
- ☕ **苦**（火）：清醒與瀉火 → 適用無法接受現實
- 🌶️ **辛**（金）：宣洩與流動 → 適用情感淤塞、憂鬱
- 🧂 **鹹**（水）：意志與記憶 → 適用失智、忘記姓名

**記憶標籤五味對照**（與 `MEMORY_FLAVOR_MAP` 一致，部分）：
- Day 1：針、縫線、寒冷、裁縫手藝、銀座的驕傲、失憶、迷茫、雨聲、潮濕、寧靜、陪伴
- Day 2：蜜糖笑容、女兒-美雪、婚紗、眼淚、美雪的笑容、第一次叫爸爸、執念、雪、死亡、最後一針、閣樓
- V4.5：失語（酸 3 / 鹹 2）、缺席的典禮（酸 1 / 苦 3 / 辛 3）、空蕩的店（酸 2 / 苦 1 / 鹹 3）
- 數值例：`裁縫手藝` 甜 3 鹹 2；`銀座的驕傲` 甜 1 鹹 3；`美雪的笑容` 甜 6；`第一次叫爸爸` 甜 5；`最後一針` 苦 3

### 縱向延伸話題系統

**設計理念**：
- 玩家視角：不知道角色背景，只能透過追問逐步挖掘
- 避免上帝視角：不直接告訴玩家「這是田中太郎，他是裁縫師」
- 層層剝開：從表層對話 → 延伸追問 → 深層記憶

**Day 1 延伸話題**：
- 「你的手」→「那…這雙手做過最驕傲的事是什麼？」（`hands_pride`）
  - 解鎖條件：完成「你的手」Part 2 後出現
  - 獲得記憶：`銀座的驕傲`（甜 1 / 鹹 3）
- 「你的手」→「既然你是裁縫，店裡生意應該很好吧？」（`twilight_artisan`）— **V4.5**
  - 獲得記憶：`空蕩的店`（職人黃昏）

**Day 2 延伸話題**：
- 「那個夢」→「夢裡那個小女孩…她小時候是什麼樣子？」（`miyuki_childhood`）
  - 獲得記憶：`美雪的笑容`、`第一次叫爸爸`
- 「那個夢」→「你說她常哭...是因為你做了什麼嗎？」（`ceremony_rift`）— **V4.5**
  - 獲得記憶：`缺席的典禮`（裂痕事件：入學典禮缺席）
- 「你怎麼來的」→「你說有很多雪…那時候你原本在做什麼？」（`snow_then`）
  - 獲得記憶：`最後一針`、`閣樓`、`執念`

### 甜/苦路線設計

**甜味過重路線**：
- 路線組成：`沉默陪伴` + `那個夢` + `關於美雪小時候`
- 聚焦溫暖記憶（美雪、陪伴、寧靜）
- 配方調整：`熱茶` 配方擴展為接受 `寧靜` + `陪伴` 記憶

**苦味過重路線**：
- 路線組成：`你怎麼來的` + `雪中那時…`（延伸話題）
- 記憶標籤：`最後一針`、`閣樓`（高苦值）
- 劇情意義：聚焦死亡前的遺憾與未完成

**結局判定邏輯**：
- 保持 `determineEnding()` 與程式一致（見上方 Day 3 表格）
- 設計理念：內容引導 > 邏輯限制
