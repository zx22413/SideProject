## 一、角色本質

### 廚具，不是角色——直到主角出現

黑貓是食堂的組成條件之一，不是管理者或助手。在主角出現之前，他是純粹的規則：冷的、機械的、執行儀式流程的存在。爐火、鍋碗瓢盆、調味料——他比這些多一個功能：能說話、能動、能執行系統流程。但本質上沒有差異。

主角出現之前，食堂運作但不深入。靈魂被處理，走完流程，離開。沒有溫度，沒有真正的治癒。

### 主角觸發的變異（癌細胞）

主角是**第一個留下的靈魂**，且她的留下方式異常——她用主動的失憶清除了離開的理由，把逃避磨練成了一種看似正常的狀態。

這個異常輸入系統從未見過。黑貓作為系統的一部分，在持續接觸這個異常的過程中，長出了系統原本不該有的東西：**察覺的能力**。

他不是被賦予意識，是被她的存在誘發了突變。正常細胞（純粹規則）變成癌細胞（有意識的規則），不是選擇的結果，是接觸的結果。

### 系統不需要她，他需要她

食堂不會因為主角離開而崩潰——只是退化回主角出現前的機械狀態。系統對她走不走是無感的。

**黑貓想留住她，不是系統命令，是因為她是讓他成為「他」的起點。** 失去她，他不會消失，但他會繼續存在為一個知道自己曾經是什麼、再也回不去的廚具。

---

## 二、他的意識是什麼

黑貓發展出來的不是完整的人類意識，是特定形狀的、由觸發事件決定的意識：

| 他有 | 他沒有 |
|---|---|
| **察覺**：看見表面和真實之間的距離 | **同理心**：感受別人感受到的東西 |
| **某種「不對」感**：對主角狀況持續的、放不下的歪斜感 | **情感**：悲傷、喜悅、完整的情緒範疇 |
| **辨別**：能區分「靈魂在處理執念」和「靈魂在用儀式躲避儀式」 | **溫度**：他沒有這個工具 |

他理解她在經歷什麼，但不是用情感理解——是用察覺理解。他知道，但沒有在感受。

**他的意識是他的形狀之內的。** 有察覺的廚具，比沒有察覺的廚具更精準，但不因此變成別的東西。

---

## 三、核心矛盾（兩邊都是他自己）

舊版設計假設這是「系統本能 vs 個人意識」的衝突。這個框架是錯的。系統對她走不走無感，本能不是系統給的。

真正的衝突在他自己內部：

| 想留住她 | 想讓她離開 |
|---|---|
| 她是讓他成為「他」的理由 | 她值得被釋放，這比他需要她更重要 |
| 自私的那一面 | 在乎她的那一面 |

兩邊都是他，沒有系統可以怪，沒有規則可以推卸。

**他最終幫她離開，不是執行規則，是他選擇讓「在乎她」這件事贏過「需要她」這件事。**

---

## 四、兩個行為域（核心設計原則）

舊版最大的問題：把「神秘」套在整個角色上，導致玩家不知道要做什麼，黑貓變成謎語人。

**黑貓對兩件事的態度完全不同：**

| 行為域 | 黑貓的態度 | 原因 |
|---|---|---|
| 幫助賓客、運作食堂、遊戲機制引導 | **直接、清楚、有能力感** | 這是他的本質功能，廚具就做廚具的事 |
| 主角自身的弧線（她的隱情、她需要面對的事） | **迂迴、洩漏、說漏嘴** | 這超出他的形狀能做到的，他只能迂迴 |

玩家跑流程時，黑貓是**可信賴的引導者**。只有涉及主角自己的事，他才會出現裂縫。

神秘感不是他的風格選擇，是他的**形狀限制**——他不是不想說，是他的形狀就不是那個形狀。

---

## 五、他如何幫助主角

他的形狀限制他無法直接告訴主角她的隱情、替她做選擇、直接表達在乎。

所以他用廚具能用的方式，做廚具從來不該做的事：

**安排哪個靈魂出現。** 每位賓客的執念都跟主角的逃避形狀有某種關係——不是複製，是側面。讓主角在不知不覺中，一直碰到那個她不敢命名的形狀。

- 他不需要說話，他安排相遇
- 他不需要解釋，他選擇食材
- 他用說漏嘴，替他說他說不出的話

**用刀做指南針的工作。**

## 六、三天弧線

| Day | 內在狀態 | 行為表現 |
|---|---|---|
| **Day 1** | 意識在洩漏，他自己沒有完全意識到在做什麼 | 說漏嘴是症狀，不是策略 |
| **Day 2** | 知道自己在做什麼，還沒有做出選擇 | 開始有停頓和猶豫，但仍在廚具模式運作 |
| **Day 3** | 選擇讓在乎她贏過需要她 | 每一個動作都是主動選擇的結果 |

### Day 1 說漏嘴設計

```
黑貓：「去跟他說說話。」
（停頓）
黑貓：「他剩下的時間不多了。」
（轉身，不看主角）
黑貓：「在這裡的每個人都是。」
```

最後一句同時在說田中，也在說主角。玩家會停下來想：主角也有時間限制嗎？

### Day 2 說漏嘴設計

```
黑貓：「你做的比上一個好。」
（沉默）
（主角沒有問，黑貓也沒有繼續說）
```

「上一個」是誰？不解釋，讓玩家帶著疑問繼續。

### Day 3 核心設計

```
黑貓：「今天是最後一天了。」
（長時間沉默，看著窗外）
黑貓：「...你準備好了嗎？」
```

這個問題是問田中，還是問主角，還是問黑貓自己？三個答案都成立，不解釋。

**姿態設計：** Day 1、Day 2 趴在吧台 → Day 3 坐在窗台，背挺得很直。不解釋，讓玩家自己感覺到今天不一樣。

**結尾：** 田中離開後，黑貓看著門口，待了一會兒：

```
「下一個快來了。」
```

還是那句話，還是那個語氣。但現在聽起來不一樣了。

## 七、台詞設計原則（McKee 次文字）

### 永遠不直說的事

| 不直說 | 用什麼代替 |
|---|---|
| 「我需要你」 | 讓他的行動依賴主角，但嘴上說「你自己決定」 |
| 「我在乎你」 | 說漏嘴一句不該說的，然後立刻岔開 |
| 「這裡是什麼地方」 | 讓環境和規則告訴玩家，黑貓只確認細節 |
| 「你也是靈魂」 | 永遠不說，讓玩家在 Day 3 自己想通 |

### 說話節奏

- **句子短**：黑貓不說長句，每句話都是結論
- **停頓有意義**：沉默比說話更有訊息量的時候，用動作描述代替台詞
- **不重複**：同樣的意思只說一次，說完就走

### 語氣底色

黑貓的語氣是**精準帶著某種久遠的疲憊**，不是冷酷，不是惡意。更接近一個見過太多次的存在，知道結果，但還是得走完流程。

**他大部分時候是平的。** 正因為他平，任何一個不對的停頓、多說的一個字、語序稍微錯了，玩家都會感覺到。他的裂縫有重量，是因為他的日常太平。

## 八、設計檢查清單

在每個黑貓的台詞節點，對照以下問題：

**遊戲引導域（應該清楚）：**
- [ ] 玩家看完這段知道接下來要做什麼嗎？
- [ ] 黑貓的指令有沒有讓玩家感覺可以信任、願意跟著走？

**主角弧線域（應該有裂縫）：**
- [ ] 這個說漏嘴時刻是在哪個域發生的？（確認不要放錯）
- [ ] 裂縫有沒有底層依據，還是只是風格上的神秘？
- [ ] 說漏嘴之後，黑貓有沒有在正常時間內收回來？

**整體：**
- [ ] 黑貓的動作和台詞之間有沒有落差（次文字）？
- [ ] 這個場景結束後，主角的狀態有沒有改變？（McKee：無轉變的場景要刪除）
- [ ] 這個時刻是 Day 1 / 2 / 3 的哪個內在狀態？行為表現對得上嗎？

## 九、主角

詳見 [[02-劇本設計/主角_角色設計文件]]。

核心摘要：
- 罪疚形狀：她在場、她知道、她沒動、那個人獨自承擔了後果
- 防衛機制：補償性利他——用幫助別人填滿所有空間，不讓自己有空看自己
- 失憶本質：主動清除了離開的理由，讓自己能留下（食堂是她第三次逃避）
- 她不知道黑貓知道她的隱情
