## 📋 角色深度檔案

### 基本資訊
- **真實姓名**: 田中太郎（たなか たろう）
- **靈魂編號**: `guest1`
- **生前年齡**: 78歲
- **生前職業**: 銀座老店「針尖工房」第三代傳人
- **專長**: 高級訂製西裝、婚紗（晚年轉型）
- **死亡時間**: 2023年12月24日，聖誕夜
- **死亡地點**: 長野縣白馬山系
- **死因**: 阿茲海默症引發的迷路 + 低溫症

### 生前故事脈絡

#### 第一階段：榮耀的職人（30-60歲）
```
1975年（30歲）：接手父親的「針尖工房」
- 在銀座三丁目經營高級西裝訂製店
- 客戶包括政商名流、演員
- 座右銘：「一針一線，不容妥協」
- 對工作極度嚴格，對家人疏於陪伴

1980年（35歲）：女兒美雪出生
- 妻子早逝（難產併發症）
- 獨自撫養女兒，但工作太忙，關係疏遠
- 美雪童年記憶：父親總在工作，很少說話

1995年（50歲）：事業巔峰
- 為首相製作訪美用西裝
- 工房名氣達到頂峰
- 但美雪已準備離家讀大學，父女關係冷淡
```

#### 第二階段：失落與轉變（60-70歲）
```
2005年（60歲）：產業衰退
- 成衣工業崛起，訂製西裝需求下滑
- 年輕人不再穿西裝，工房生意驟降
- 開始懷疑「一生堅持的技藝是否還有意義」

2010年（65歲）：女兒結婚通知
- 美雪打電話：「爸，我要結婚了。」
- 田中：「...恭喜。需要我做什麼嗎？」
- 美雪：「不用了，婚紗我已經訂好了。」
- 電話掛斷，田中看著空蕩的工房

【轉折點】
- 當晚，田中翻出妻子留下的婚紗設計草圖
- 決定：「我要為美雪親手縫製婚紗」
- 這是他第一次主動為女兒做些什麼
```

#### 第三階段：贖罪與遺忘（70-78歲）
```
2015年（70歲）：秘密計畫
- 開始學習婚紗製作（他從未做過女裝）
- 白天維持工房營運，夜晚在閣樓縫製
- 選用最好的絲綢，使用母親留下的蕾絲
- 目標：在美雪30歲生日送給她

2018年（73歲）：失智初期
- 開始忘記客人名字
- 偶爾找不到工具
- 但婚紗進度持續推進，這是他唯一記得清楚的事

2020年（75歲）：疫情與孤獨
- 工房因疫情關閉
- 美雪因工作繁忙，一年只來看一次
- 田中幾乎將所有時間投入婚紗
- 病情加重：常常忘記今天是星期幾

2023年12月（78歲）：最後一針
- 婚紗只剩最後一個釦眼未完成
- 12月24日，聖誕夜
- 田中在閣樓工作，突然「忘記自己在做什麼」
- 困惑地穿上大衣走出門
- 「我要去哪裡？我在找什麼？」
- 迷失在白馬山系的雪地中
- 手中緊握著那根銀針
```

### 核心創傷的三層結構

| 層次 | 認知 | 情感 |
|------|------|------|
| **表層**（玩家初見） | 我在雪山中凍死了 | 寒冷、迷茫、恐懼 |
| **中層**（Day 1-2） | 我忘記了重要的事 | 焦慮、懊悔、空虛 |
| **深層**（Day 3） | 我未能完成對女兒的愛 | 自責、遺憾、渴望救贖 |

### 五味診斷（中醫心理學）

| 風味 | 對應情緒 | 治療作用 | 在田中身上的體現 |
|------|---------|---------|---------------|
| **鹹**（水/腎） | 失憶、恐懼 | 喚醒深層記憶 | 他忘記了婚紗，忘記了女兒 |
| **甜**（土/脾） | 寒冷、匱乏 | 撫慰與接地 | 他在雪中凍死，需要溫暖 |
| **苦**（火/心） | 執著、後悔 | 清醒面對真相 | 他執著於「一針一線」，忽略了陪伴 |

**完美組合**: 鹹（記憶）+ 甜（愛）+ 苦（覺醒）= 蜜汁炙燒鹹魚
