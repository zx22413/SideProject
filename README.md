> **專案代號**：CONFITEOR  
> **當前版本**：Ink 轉型期  
> **專案狀態**：劇情以 Ink 為 SSOT；現有內容自 GAS 翻譯中，其後於 Ink 內寫完剩餘劇情，再選殼（Twine／網頁／Ren'Py）  
> **專案類型**：Ink 驅動之文字敘事遊戲（載體中立）

## 專案簡介

**靈魂食堂**是一款以**劇情可編輯為核心**的文字敘事專案。玩家扮演「罪名吞噬者（Sin-Eater）」，經營一間位於生死交界的邊境食堂，透過五味診斷與料理救贖幫助困於執念的靈魂超渡。

### 核心特色
- **五味考古系統**：中醫五行理論映射情緒與記憶
- **標籤驅動敘事**：玩家選擇影響劇情分支，收集標籤解鎖隱藏劇情
- **儀式結構**：誘捕・偵訊・赦免，故事分段落推進、單次可完結
- **Ink 為劇情 SSOT**：先寫故事、後跑起來；內容可無縫銜接不同殼（網頁、Twine、Ren'Py 等）
- **情感優先設計**：參考《Bird Alone》《Bear's Restaurant》的敘事設計

## 核心機制簡述

### 五味考古系統
| 五味 | 五行 | 遊戲作用 |
|------|------|---------|
| 甜 | 土 | 撫慰與接地 |
| 酸 | 木 | 聚焦與收斂 |
| 苦 | 火 | 清醒與瀉火 |
| 辣 | 金 | 宣洩與流動 |
| 鹹 | 水 | 意志與記憶 |

### 儀式流程
```
誘捕（Lure）→ 收集感官標籤，靈魂樣貌逐漸清晰
偵訊（Feast）→ 標籤觸發記憶場景，解鎖目標料理
赦免（Absolution）→ 烹飪 → 記憶重現 → 告別 → 獲得遺物
```
