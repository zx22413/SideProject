# 輸入形式 (Input Forms)

> [!info] 快速連結
> 詳細輸入載體請見：[[04-資源素材/輸入載體]]

## 快速摘要

**輸入形式**是收集用戶回饋與驗證資料的各種載體形式。

### 核心形式

根據 [[驗證方式以及回饋模型]]，主要輸入形式包括：

1. **[[驗證方式-Demo]]**：視覺化 Demo（Vibe Code 製作的情緒鏡子）
2. **[[驗證方式-實驗]]**：7 天精靈對話實驗（Wizard of Oz）
3. **[[驗證方式-聊天]]**：與特定朋友的 Coffee Chat
4. **[[驗證方式-訪談]]**：深度訪談
5. **[[驗證方式-問卷]]**：結構化問卷調查
6. **[[驗證方式-文章]]**：文章發表（僅作為整理思緒用）
