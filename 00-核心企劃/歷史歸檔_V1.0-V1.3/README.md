## 🔄 為何這些企劃被歸檔？

### 問題診斷（參考 [[../../00-工作區/架構診斷報告_2026-01-15]]）

1. **產品定位裂縫**：「電子寵物包裝的日記」造成期待落差
   - 用戶期待寵物的視覺成長 vs. LINE 只有文字對話
   - 用戶期待日記的回顧功能 vs. LINE 無法有效搜尋

2. **載體不匹配**：LINE OA 不適合「養成遊戲」
   - Rich Menu 無法呈現寵物狀態
   - 對話流是線性的，無法展示累積感

3. **成本結構問題**：7 日推播機制費用過高
   - 每日主動推播 = 計費訊息
   - 改為回合制 = 完全免費

### 轉型決策

最終決定將產品定位從「電子寵物」轉型為「敘事遊戲」：
- **新核心**：料理喚起記憶 → 靈魂超渡
- **新機制**：3 日考古 → 回合制互動
- **新載體優勢**：LINE 的文字介面完美適合劇情推進
