## 🎯 專案核心概念

### 世界觀
玩家扮演「罪名吞噬者（Sin-Eater）」，經營一間位於生死交界的「邊境食堂」。這裡接待因執念而困於時間的靈魂，透過料理喚起記憶，助其解脫前往來世。

### 核心機制：五味考古系統

| 五味 | 五行 | 治療情緒 | 敘事功能 |
|------|------|---------|---------|
| 甜 | 土 | 焦慮 | 撫慰與接地 |
| 酸 | 木 | 悔恨 | 聚焦與收斂 |
| 苦 | 火 | 執念 | 清醒與瀉火 |
| 辣 | 金 | 悲傷 | 宣洩與流動 |
| 鹹 | 水 | 失憶 | 意志與記憶 |

### 三日儀式流程
1. **Day 1（誘捕）**：透過互動收集感官標籤（溫暖、柔軟、鹹味等）
2. **Day 2（偵訊）**：標籤組合觸發記憶場景，解鎖目標料理
3. **Day 3（赦免）**：烹飪料理 → 記憶重現 → 告別儀式 → 獲得遺物

## 🔑 設計核心原則

### 三大不變的底層規則

1. **低成本、非同步互動**
   - 採用回合制，玩家主動觸發 → Reply API 免費
   - 跨日鎖定機制 → 強制懸念與想念

2. **高情感投入**
   - 3 日濃縮 Bird Alone 的情感曲線
   - 「見證而非審判」的敘事範式

3. **文字作為核心媒介**
   - 標籤驅動對話系統
   - 對話選擇影響劇情分支

## 🚨 與舊版本的關鍵差異

| 維度 | 雲寶（V1.0-V1.4） | 靈魂食堂（V4.0） |
|------|------------------|------------------|
| **定位** | 電子寵物 + 日記 | 敘事遊戲 |
| **版本代號** | Cloudy | CONFITEOR |
| **時長** | 7 日 | 3 日 + 跨日鎖定 |
| **角色** | 單一寵物（雲寶） | 多位靈魂（Guest 系統） |
| **互動** | 餵食 + 聊天 | 五味診斷 + 標籤考古 |
| **目標** | 陪伴養成 | 救贖超渡 |
| **重玩性** | 低（劇情重複） | 高（每位靈魂不同） |
| **數據結構** | 3 欄 | 10 欄 + 4 輔助 Sheets |
