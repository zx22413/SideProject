### 儀式化循環
🌙 Night（觀察）→ 🌅 Day（對話）→ 🍳 Cooking（料理）→ 🌃 After（揭露）

- 去說教化：黑貓不解釋，玩家自己體會
- 去中二化：用主廚/食堂/客人，不用「罪惡吞食者」
- 延遲揭露：Day 3 才揭曉真相
- 台詞生動化：正常對話流暢，記憶閃現碎片化
