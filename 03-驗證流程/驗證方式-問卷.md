# 📝 問卷設計：雲寶 Beta 封測回饋單 (v2.0)

## 問卷架構 (Google Form)

### 【第一部分：飼養員檔案】
**Q1. 你的生理性別是？** (驗證風格受眾)
* 男生 / 女生 / 其他

**Q2. 你目前的身份狀態是？ (主要壓力來源)**
* 學生 / 社會新鮮人 / 資深工作者 / 自由業 / 待業中

**Q3. 遇到雲寶前，心情不好通常怎麼處理？**
* 找人講 / 寫日記 / 發洩轉移 / 悶在心裡 / 跟 AI 聊

### 【第二部分：遊戲體驗】
**Q4. 這次測試中，你對雲寶目前的「幼兒/呆萌」語氣感覺如何？**
* 超級喜歡 / 還可以 / 太吵太黏 / 不喜歡

**Q5. 請誠實告訴我，你撐到第幾天？**
* Day 1-2 / Day 3-6 / Day 7 結局 / 超過 Day 7

**Q6. (如果沒玩完) 讓你棄坑或忘記開的原因是？**
* 沒通知 / 不知講啥 / 回應單一 / 太花時間 / 其他

### 【第二部分-1：每日選擇題機制 (V1.2 新增)】
**Q6-1. 每日選擇題（例如「咖哩怎麼吃？A/B」）是否有幫助你開啟對話？**
* 非常有幫助 / 有點幫助 / 沒什麼幫助 / 反而造成壓力

**Q6-2. 你覺得選擇題的難度如何？**
* 太簡單 / 剛好 / 有點難 / 太難

**Q6-3. 你願意回答這些選擇題嗎？**
* 每天都會回答 / 大部分時候會 / 偶爾會 / 不太想回答

**Q6-4. 你覺得選擇題的設計（A/B 二選一）比開放式問題更容易回答嗎？**
* 容易很多 / 稍微容易 / 差不多 / 反而更難

### 【第三部分：未來許願池】
**Q7. 如果雲寶要升級，你「最希望」先開發哪個功能？(複選前3名)**
* 📸 專屬劇情圖片 (看到雲寶長手的照片)
* 🧠 可調教性格 (高冷/溫柔/毒舌)
* 🎲 更多隨機事件 (撿到寶物、生病)
* 👾 不同的寵物種類 (雷電球、水滴怪)
* 🤝 社交互動 (去朋友家玩)

**Q8. 承上題，如果「生成專屬圖片」需要付費，你願意嗎？**
* 願意付錢 / 只願看廣告 / 不願意

### 【第四部分：平台與付費】
**Q9. 如果雲寶要正式搬家，你最希望它住在哪裡？**
* LINE 官方帳號 / 獨立 App / 桌面 Widget / 維持 ChatGPT
